Build Your Own Zero Footprint Web Viewer
========================================

Pre-requisities
---------------

* [WebStorm](https://www.jetbrains.com/webstorm/download/)

* [Node.js](www.node.js)

* [Grunt](http://gruntjs.com/)

> npm install -g grunt-cli

* [Bower](http://bower.io/)

> npm install -g bower
